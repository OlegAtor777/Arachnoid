# 🕷️ Arachnoid — Bluetooth-управляемый паукообразный робот на ESP32 с FreeRTOS

**Arachnoid** — это 3D-печатный четырёхногий робот (spider bot), построенный на базе **ESP32**, с управлением по Bluetooth-геймпаду и многозадачной архитектурой на **FreeRTOS**.

За основу для 3D моделей взят проект "Quadruped Spider robot" от CirquitDigest: https://www.thingiverse.com/thing:6182849

Проект демонстрирует практическое применение:

- Embedded-разработки на ESP32
- Параллельных задач и синхронизации в FreeRTOS
- Алгоритмической анимации походки
- Управления сервоприводами в реальном времени
- Интеграции Bluetooth-геймпада через библиотеку Bluepad32
- Мехатроники и 3D-проектирования корпуса

Репозиторий: GitHub

## 🚀 Ключевые возможности

- 🎮 Полноценное управление роботом с Bluetooth-геймпада
- 🧠 Многозадачная архитектура (3 FreeRTOS задачи на двух ядрах ESP32)
- 🦿 Диагональная походка с фазированием ног
- ⚙️ Плавная синусоидальная анимация движения
- 🏎️ «Sport mode» с динамическим изменением скорости
- 🔁 Зеркальная кинематика для левых и правых ног
- 🧵 Потокобезопасная работа через mutex

## 🧠 Архитектура системы

Проект разделён на три независимые задачи FreeRTOS:

| Задача | Ядро | Приоритет | Назначение |
|--------|------|------------|------------|
| GamepadTask | Core 0 | Высокий | Чтение данных с геймпада и управление направлением |
| WalkingTask | Core 0 | Средний | Логика включения/выключения ходьбы |
| ServoAnimationTask | Core 1 | Низкий | Плавная анимация движения сервоприводов |

Такое разделение:
- исключает блокировки
- обеспечивает отзывчивость управления
- разгружает основное ядро
- демонстрирует грамотную работу с многопоточностью на микроконтроллере

## 🕹️ Управление с геймпада

| Действие | Эффект |
|----------|--------|
| D-Pad ↑ | Движение вперёд |
| D-Pad ↓ | Движение назад |
| D-Pad ← | Движение влево |
| D-Pad → | Движение вправо |
| Правый стик ←/→ | Поворот на месте |
| Кнопка A | Переключение Sport Mode (ускорение походки) |

---

## 🦿 Алгоритм походки

В основе лежит фазированный цикл из `ANIMATION_STEPS = 60` шагов.

- Ноги работают по диагональным парам
- Для каждой ноги задан сдвиг фазы
- Горизонтальное движение рассчитывается через `sin()`
- Вертикальное — через `cos()`
- Это создаёт естественную, плавную, непрерывную походку без рывков

```cpp
horizontal = sin(phase)
vertical   = cos(phase)
```

Зеркальность левых и правых ног достигается изменением направления углов без изменения формулы движения.

## ⚙️ Аппаратная часть

### Компоненты

- ESP32
- 8 × сервоприводов (SG90 / MG90S)
- Внешнее питание сервоприводов
- Bluetooth-геймпад (DualShock, Xbox, 8BitDo и др.)
- 3D-печатный корпус (STL в репозитории)

### Подключение сервоприводов

| Нога | Горизонтальный | Вертикальный |
|------|------------------|---------------|
| Передняя правая | GPIO 21 | GPIO 19 |
| Задняя правая   | GPIO 33 | GPIO 25 |
| Задняя левая    | GPIO 27 | GPIO 14 |
| Передняя левая  | GPIO 12 | GPIO 13 |

## 🧵 Потокобезопасность

Для синхронизации используется `SemaphoreHandle_t mutex`.

Все критичные переменные:
- направление движения
- флаг ходьбы
- режим скорости

изменяются только внутри защищённых секций.

Это предотвращает race condition между задачами.

---

## 🏎️ Режимы скорости

Переключаются кнопкой A:

| Режим | Интервал шага |
|-------|----------------|
| Обычный | 25 мс |
| Sport | 15 мс |

Изменяется частота анимации без изменения амплитуды движений.

## 🛠️ Как запустить

1. Клонировать репозиторий:
 ```bash
 git clone https://github.com/OlegAtor777/Arachnoid.git
 ```
2. Открыть `.ino` файл в Arduino IDE
3. Установить библиотеки:
 - Bluepad32
 - ESP32Servo
4. Выбрать плату ESP32
5. Загрузить прошивку
6. Подключить геймпад по Bluetooth
7. Управлять роботом
